فئات التحسين الرئيسية

Largest Contentful Paint (LCP)

1. تحديد وتحسين الصورة/العنصر الأكبر (LCP Element)
2. تقليل زمن استجابة الخادم (TTFB) إلى أقل من 600 مللي ثانية

Cumulative Layout Shift (CLS)

3. تحديد أبعاد (width/height) جميع الصور والفيديوهات في HTML
4. تجنب إدخال محتوى فوقي (Above-the-fold) بشكل ديناميكي (مثل الإعلانات)

First Input Delay (FID) / INP

5. تقليل حجم ملفات JavaScript وإزالة الأكواد غير المستخدمة (Tree-shaking)
6. تقسيم الكود (Code Splitting) وتحميل المكونات عند الطلب (Lazy Loading)

تحسينات عامة وأساسية

7. تفعيل ضغط ملفات الخادم (Gzip أو Brotli)
8. تصغير ملفات CSS و JavaScript (Minification)
9. استخدام صيغ صور حديثة (WebP/AVIF) وتحميل متأخر (Lazy Load)
10. استخدام شبكة توصيل المحتوى (CDN) لخدمة الأصول الثابتة

مُقدمة بواسطة تاج نوفا - مجانية للمطورين.
